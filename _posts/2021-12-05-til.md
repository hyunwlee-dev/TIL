<span style="background-color:#85B8CB; color:white"><strong>풀이</strong></span>

#### <u>0은 1로부터 온다.</u> 

#### 1은 0과 2로 부터 온다. 2는 1과 3으로 부터 온다... 8은 7과 9로 부터 온다. 

#### <u>9는 8로 부터온다.</u>

```
// pseudo code

/* init */
memo[1][0]  ~ memo[1][9]= 1;

/* recurse */
if (i == 0)
	return (D[N][i] = D[N - 1][1]);
if (i == 9)
	return (D[N][i] = D[N - 1][8]);
return (D[N][i] = D[N - 1][i - 1] + D[N - 1][i + 1]);
```
